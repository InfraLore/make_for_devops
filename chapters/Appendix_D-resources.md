# Appendix D: Resources

## Make Cheat Sheet

A standalone printable PDF cheat sheet covering Make syntax, common patterns,
debugging tips, and more is maintained as a separate project:

<https://github.com/InfraLore/make_cheat_sheet>

Build it locally with `make build`, or download the latest PDF from the
releases page.

## Official Documentation

- [GNU Make Manual](https://www.gnu.org/software/make/manual/)
- [Pandoc Manual](https://pandoc.org/MANUAL.html)

## Recommended Reading

- *Managing Projects with GNU Make* (O'Reilly) — the canonical reference
- [Makefilet](https://github.com/jhriv/makefilet) — John H. Robinson IV's
  Makefile tutorial collection
- [Self-Documenting Makefile pattern](/makefiles/self-documenting-makefile/) —
  the awk-based help system used throughout this book
